# westyle | Streetwear Luxury Drop ⚡

An immersive, high-end minimalist E-commerce landing page tailored for modern luxury streetwear collections. Built with meticulous precision to offer a seamless retail browsing experience featuring robust state management, ultra-clean UI aesthetics, a dynamic theme switching engine, and a fully automated WhatsApp checkout pipeline.

Live Preview available at: **[michaelsamy210.github.io/westyle-luxury-store](https://michaelsamy210.github.io/westyle-luxury-store/)**

---

## ✨ Features & Architecture Highlights

- **Dynamic Theme Switcher**: Fluid transitions between a high-contrast Dark Mode and a sleek Light Mode using pure CSS Custom Properties (`--variables`).
- **Interactive Cursor-Glow Tracker**: Immersive backdrop illumination engine that tracks mouse telemetry seamlessly across viewport boundaries to elevate visual engagement.
- **Advanced State Management Grid**: Lightweight vanilla JavaScript shopping cart architecture ensuring dynamic calculations for item sizes, colors, quantitative incrementation, and real-time interface sync.
- **Secure Financial Logic Matrix**: Automated coupon valuation pipeline supporting dynamic coupon verification (e.g., `WESTYLE10`) alongside adaptive geographical shipping rate allocations based on customer destination.
- **Polished WhatsApp Checkout Manifest**: Seamlessly compiles and structure user checkout records (Name, Phone, Address, Cart Items, Pricing) into a clean, professional URL-encoded layout to transmit directly to the merchant hub.
- **Fully Responsive Architecture**: Meticulously scaled utilizing CSS media-query boundaries to guarantee an optimal aesthetic presentation from widescreen 4K displays down to standard modern smartphone devices.

---

## 🛠️ Built With

- **HTML5**: Structured semantic markers engineered for clarity and modern layout stability.
- **CSS3**: Modular layout design built utilizing flexbox/grid mechanics, smooth transition matrices, and absolute root custom property definitions.
- **Vanilla JavaScript (ES6+)**: High-performance client-side logic completely unburdened by heavy, unnecessary third-party frameworks.
