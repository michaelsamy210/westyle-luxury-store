# 📊 Advanced Ad Performance Calculator

A specialized, high-performance web application designed for digital marketers and media buyers to accurately evaluate the financial health of their advertising campaigns.

## ✨ Features
* **Dual Metrics Calculation:** Computes both Return on Ad Spend (ROAS) and total Return on Investment (ROI) simultaneously.
* **Advanced Financial Logic:** Accounts for product COGS, shipping, and hidden overhead costs for accurate ROI.
* **Dynamic UI Feedback:** Real-time client-side rendering that changes UI colors dynamically (Green for profitable campaigns, Red for net losses).
* **Defensive Architecture:** Built with robust JavaScript input validation to prevent zero or negative value errors.
* **Premium Glassmorphic Design:** Sleek modern dark mode UI that scales beautifully across mobile and desktop devices.

## 🛠️ Tech Stack
* **HTML5:** Semantic architecture for better accessibility.
* **CSS3:** Custom layout styling, CSS variables, and glassmorphism.
* **Vanilla JavaScript:** DOM manipulation and core financial logic (ES6+).
